A comprehensive Python tutorial demonstrating how to collect, process, analyze, and visualize scientific publication data using the IEEE Xplore API. Through a real-world case study based on IEEE ICASSP publications, this repository presents a complete, reproducible workflow that transforms raw publication metadata into meaningful insights, publication-quality visualizations, animations, and interactive dashboards.

Developed in collaboration with the **BRICS+EU Think Tank** and the **BRICS+ Audio, Acoustics and Music Technology Group**, this notebook is intended for students, early-career researchers, educators, and practitioners interested in learning practical data acquisition, data analysis, bibliometric analysis, and scientific visualization with Python.

## Features

- Automated data acquisition from IEEE Xplore
- Web automation using Playwright
- Large-scale metadata collection and processing
- Country affiliation extraction and normalization
- Data cleaning and validation
- Exploratory data analysis with Pandas
- Scientific visualization using Matplotlib
- Multi-layer donut chart development
- Animated visualizations of publication trends
- Interactive dashboards using Chart.js
- JSON-based analytical workflows
- Comparative analysis of international research activity

## Learning Outcomes

By working through this notebook, you will learn how to:

- Collect publication metadata from IEEE Xplore
- Automate web-based data collection workflows
- Clean, validate, and organize large datasets
- Perform exploratory data analysis using Pandas
- Create publication-quality scientific visualizations
- Develop interactive dashboards
- Build reproducible analytical workflows in Python
- Transform raw research data into actionable insights

## Case Study

The tutorial uses publication data from the **IEEE International Conference on Acoustics, Speech, and Signal Processing (ICASSP)** to demonstrate modern bibliometric analysis techniques. Beyond the technical workflow, the analysis explores changes in the global signal processing research landscape, highlighting the increasing contribution of BRICS+ countries and the evolving international distribution of scientific publications.
